ሕጋዊ አኀዝ (Ordinal Numbers)
ኣሐድ / ቀዳማይ = 1ኛ
ካልእ / ካልዐይ = 2ኛ
ሣልስ / ሣልሳይ = 3ኛ
ምሳሌ፡- ቀዳማይ ዘመን
መጀመሪያው ዘመን
፪
ክፍላዊ አኀዝ (Fractional Numbers)
ካልዒት = ½ (ግማሽ)
ሣልሲት = ⅓ (ሲሶ)
ራብዒት = ¼ (ሩብ)
ምሳሌ፡- ካልዒተ ክልዔቱ
ሁለት ሁለተኛ (ግማሽ)
፫
ዕፅፋዊ አኀዝ (Multiplicative Numbers)
ካዕብተ ክልዔቱ = 2x (ሁለት እጥፍ)
ካዕብተ ሰለስቱ = 3x (ሦስት እጥፍ)
፬
መድበላዊ አኀዝ (Indefinite Numbers)
ብዙኅ = ብዙ
ንስቲት = ትንሽ
ኅዳጥ = ጥቂት
፲፩
ውሁድ ቁጥሮች (Compound Numbers)
፲፩ = 11 (ዐሠርቱ ወአሐዱ)
፲፪ = 12 (ዐሠርቱ ወክልዔቱ)
፲፫ = 13 (ዐሠርቱ ወሠለስቱ)
፳፩ = 21 (እሥራ ወአሐዱ)
፻፩ = 101 (ምዕት ወአሐዱ)
፻፳፩ = 121 (ምዕት ወእሥራ ወአሐዱ)
